About This Project

AI-Powered Carbon Footprint Analyzer

CarbonLens bridges AI, data analytics, and sustainability. It empowers individuals to understand their carbon impact through a seamless pipeline from data collection to machine learning predictions and Power BI visualization.

✦

Domain: Artificial Intelligence & Data Analytics

✦

ML Models: Linear Regression · ARIMA · Random Forest for emission classification

✦

Data Flow: CSV Upload → Flask API → ML Model → Dashboard → Certificate

✦

Goal: Enable data-driven climate action through intelligent analytics.
